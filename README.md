# goxel

-----------------

## Description

	goxel like axel tool

## Installation

	$ go get github.com/WayneZhouChina/goxel

## Usage

	goxel [options...] <url>
	
	Options:
		-n  Numbers of blocks to run.
		-H  Add header string.
		-U  Set user agent.
		-v  More status information.
		-O  Specify local output file.
		-h  This information
		-V  Version information
