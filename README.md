# goxel

-----------------

## Description

	goxel like axel tool

## Installation

	$ go get github.com/WayneZhouChina/goxel

## Usage

	goxel [options...] <url>
	
	Options:
		-n  Numbers of blocks to run[must].
		-H  Add header string.
		-v  More status information.
		-o  Specify local output file[must].
		-h  Help information.
		-V  Version information.
